# Habit Tracker

一个口语练习打卡应用，支持文字和图片打卡。

## 架构

- **前端**: Vue 3 + Vite → GitHub Pages
- **后端**: Express → Vercel

## 本地开发

```bash
# 启动后端
cd backend
npm install
npm start

# 启动前端
cd frontend
npm install
npm run dev
```

## 部署

- 前端: 自动部署到 GitHub Pages
- 后端: 点击 Deploy 按钮部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)
